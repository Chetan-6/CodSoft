- Python
- Pandas
- NumPy
- Scikit-learn

## 📊 Approach
- Data Preprocessing
- Handling Imbalanced Data
- Feature Scaling
- Model Training (Logistic Regression / Random Forest)

Successfully identified fraudulent transactions with good accuracy and recall.
